## 🔍 样例查询

在 Neo4j Browser 中执行:

```cypher
// 找一个角色的完整关系网
MATCH (c:Character {name_en: 'Spider-Man'})-[r]-(connected)
RETURN c, type(r), connected

// 看看哪个团队人最多
MATCH (c:Character)-[:成员]->(t:Team)
RETURN t.name, count(c) AS members ORDER BY members DESC

// 查关联密度
MATCH (c:Character) WHERE size((c)-[]-()) > 10
RETURN c.name, size((c)-[]-()) AS connections ORDER BY connections DESC
```
